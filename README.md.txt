# Проект автоматизации модуля Windows (WorldSkills Almaty)

Репозиторий содержит программные решения для развертывания и автоматизации инфраструктуры сети **alma.local** и **asta.alma.local** согласно конкурсному заданию.

## Члены проектной команды и авторы
* **Разработчик/Администратор:** Марат Ермахан

## 📁 Структура проекта
* `create_user.ps1` — Скрипт автоматизации для массового создания пользователей Active Directory (MKT, SALES, TECH, HR) с поддержкой аргументов и проверок.
* `manage-shares.yaml` — Ansible-плейбук для создания сетевых ресурсов (шар) на серверах FILE-SRV и DC2, а также распределения прав доступа NTFS.

## ⚙️ Технические характеристики планируемой сети
* **Домен верхнего уровня:** alma.local (DC1: 10.10.0.10)
* **Дочерний домен:** asta.alma.local (DC2: 192.168.128.10)
* **DHCP Сеть:** 172.16.0.0/12 (Аренда на 8 дней 12 часов 25 минут)
